# AI Career Showcase — Demo

A stylish static demo website to present your work experience, projects, and dashboards with AI-inspired animations.

## Run locally

Recommended (Node.js installed):

1. Open a terminal in this folder (`c:\workspace\resume_web`).
2. Run a simple static server (no install required):

   npx http-server -p 5173

3. Open http://localhost:5173 in your browser.

Alternative (Python):

python -m http.server 5173

Enjoy! ✨
